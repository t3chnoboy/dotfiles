Examples: facebook.com, google.com/reader

Be aware that domain filters can overlap. Example: if you have filtered "picasaweb.google.com" and "google.com", removing "picasaweb.google.com" will not reactivate Picasa Web Albums, as it will still be excluded by the "google.com" filter.

Support Hover Zoom Development

Hover Zoom is distributed for free and is supported via affiliate links. You can show your support to the project by keeping this option enabled, or you can disable it.

What will happen if I enable affiliate links?
Some ads will be displayed amongst Facebook and YouTube regular ads every once in a while. These ads were designed to be unobtrusive and won't degrade your user experience. Hover Zoom does not block or replace existing ads.

Note: affiliate links won't be displayed if you use an ad blocking extension such as AdBlock.

Will I lose functionality if I disable affiliate links?
No, enabling affiliate links is only a way to show your support to the project's development.
